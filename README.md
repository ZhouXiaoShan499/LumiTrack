# LumiTrack

> 基于 **ESP32-S3-EYE** 开发板（芯片 ESP32-S3）的 LVGL 专注计时 UI 应用。

---

## 硬件平台

| 项目 | 规格 |
|------|------|
| 开发板 | **ESP32-S3-EYE** |
| 芯片 | ESP32-S3 |
| LCD | SPI 接口显示屏 |
| 传感器 | QMA6100P 加速度计（自动旋转） |
| 按键 | 实体按键（手动旋转） |

---

## 软件开发环境

| 工具 | 版本 |
|------|------|
| ESP-IDF | 5.4+ |
| LVGL | 9.5 |
| BSP | esp32_s3_eye (v6.0.0) |
| 编译器 | xtensa-esp32s3-elf-gcc 14.2.0 |
| FreeRTOS | ✅ |

## 当前实现

### LVGL UI 界面

```
╔════════════════╗
║   LumiTrack    ║
╠════════════════╣
║ 当前状态       ║
║               ║
║ 📖 看书        ║
║               ║
║ 专注时间       ║
║               ║
║ 01:28:46      ║
║               ║
║ 今日累计       ║
║ 04:12:00      ║
╚════════════════╝
```

- **LumiTrack** 标题栏
- **当前状态** — 显示当前活动（看书/学习/看电脑/看手机）
- **专注时间** — 实时计时器，精确到秒
- **今日累计** — 今日总专注时长统计

### 传感器功能

- **QMA6100P 加速度计** — 自动检测设备方向，实时旋转屏幕
- **按键控制** — 支持手动旋转（左/右）
- **方向检测** — 识别顺时针/逆时针旋转

## 快速开始

```bash
# 设置目标芯片
idf.py set-target esp32s3

# 编译
idf.py build

# 烧录
idf.py -p COMx flash monitor
```
